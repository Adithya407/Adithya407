### 🛰️ Flagship Project

<table>
  <tr>
    <td>
      <b>SAR + MSI Fusion Pipeline</b><br/>
      A SAR-guided deep learning framework to super-resolve multispectral imagery using CNNs or Transformers. Learns cross-domain correlations between radar backscatter and optical reflectance. Uses InSAR-derived DEMs for 3D terrain reconstruction.<br/><br/>
      <b>Pipeline:</b> SAR Preprocessing → MSI Preprocessing → Coregistration → InSAR/DEM Generation → Feature Extraction → DL Fusion + Super-Resolution → 3D Reconstruction → Downstream Classification → Validation<br/><br/>
      <b>Applications:</b> Urban mapping · Infrastructure monitoring · Land-cover classification · Post-disaster assessment
